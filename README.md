# Synapse - Collaborative Ideation Platform

## Overview

Synapse is an innovative iOS social media application designed for collaborative ideation, project incubation, and structured teamwork. It serves as a dedicated platform where ideas grow into projects, fostering creativity, productivity, and shared accomplishment.

## Core Features

### 🚀 Idea Sparks
- Submit and discover creative ideas
- Add tags for better discoverability
- Public and private idea sharing
- Like and comment on ideas

### 👥 Incubation Pods
- Convert ideas into collaborative projects
- Real-time team communication
- Task management and progress tracking
- Member roles and permissions

### 🔍 Discovery & Collaboration
- Explore public ideas and pods
- Search by keywords, tags, or users
- Request to join public pods
- Find like-minded collaborators

### 📱 User Experience
- Clean, modern interface with encouraging design
- Real-time notifications
- Profile management with skills and interests
- Progress tracking and analytics

## Design System

### Colors
- **Primary Green**: #3D9970 (Growth & Success)
- **Accent Orange**: #FFB347 (Creativity & Energy)
- **Accent Blue**: #87CEEB (Calm & Focus)
- **Text Primary**: #333333 (Deep Charcoal)
- **Text Secondary**: #666666 (Slate Gray)
- **Background**: #FFFFFF (Clean White)
- **Background Secondary**: #F8F8F8 (Pale Gray)

### Typography
- Modern sans-serif fonts (SF Pro as fallback)
- Clear visual hierarchy
- Readable and welcoming feel

## Technical Stack

- **Language**: Swift
- **Framework**: SwiftUI
- **Backend**: Firebase (planned)
  - Firestore for real-time database
  - Firebase Authentication
  - Firebase Cloud Storage
- **Architecture**: MVVM with async/await

## Project Structure

```
Synapse/
├── SynapseApp.swift          # Main app entry point
├── ContentView.swift         # Tab-based navigation
├── Colors.swift             # Custom color definitions
├── Models.swift             # Data models
├── ExploreView.swift        # Idea discovery feed
├── CreateIdeaView.swift     # Idea submission form
├── MyPodsView.swift         # User's incubation pods
├── NotificationsView.swift  # User notifications
└── ProfileView.swift        # User profile and settings
```

## Getting Started

1. Open the project in Xcode
2. Build and run on iOS Simulator or device
3. Explore the different tabs and features
